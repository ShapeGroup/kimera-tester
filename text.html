❯❯ Print a preformatted text:

Pre it's not valid tag for codes.
For exemple, pre, can't print a script tag.

Basically no tags can contain code.
We suggest:
 - Use PRE tag for prepackaged texts
 - use CODE tag with the dynamic system for a block of code.
 - use SAMP tag (if possible with the dynamic system) for a partial printing.
