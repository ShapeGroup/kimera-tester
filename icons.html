❯❯ Animated icon vs framework:

The framework does not natively support font icons elements or external svg presets. However, you can integrate any svg from any source ... as long as you reset it. Exemple: Get Icon from astrid or fontawesome, clean it (width,height, and other...) apply new css animation based on ex.

Note: Buttons have a better custom img positioner for static icons.
